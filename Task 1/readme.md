# Simple Calculator

A minimal command-line calculator implemented in Python.

Features
- Addition, subtraction, multiplication, division
- Input validation and a safe divide-by-zero message
- Interactive menu with an exit option

Files
- [Task 1/calculator.py](Task 1/calculator.py) — main program implementing the calculator

Key symbols
- [`calculator.add`](Task 1/calculator.py)
- [`calculator.subtract`](Task 1/calculator.py)
- [`calculator.multiply`](Task 1/calculator.py)
- [`calculator.divide`](Task 1/calculator.py)
- [`calculator.operations`](Task 1/calculator.py)

Requirements
- Python 3.6+

Running
1. Open a terminal in the project directory (the folder that contains `Task 1`).
2. Run:
```sh
python "Task 1/calculator.py"


Usage

1. Choose an operation by entering a number (1–4).
2. Enter two numbers when prompted.
3. Choose 5 to exit.

Example session:

--- Simple Calculator ---
1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Exit
Enter your choice: 1
Enter the first number: 4
Enter the second number: 5
